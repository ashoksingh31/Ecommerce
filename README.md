# Ecommerce Store API

## Tech Stack

- Python
- FastAPI
- Pydantic
- Pytest

## Features

- Add items to cart
- View cart
- Checkout
- Generate discount coupons
- View orders
- View statistics

## Run

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```
